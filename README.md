## 環境
 * Unity2019 1.3f.1

## 2DAnimation
こちらのサンプルを確認するためには前提として、Unity PackageManagerから「2DAnimation」、「2D PSD Importer」をインストールをする必要があります。

### サンプル
Unity_chan.unitypackageがプロジェクト上にインポートをしてください。

packageは次の構成をされています。
![image](https://user-images.githubusercontent.com/319724/62834874-0150e480-bc8d-11e9-8dc3-f8ed4d940c5f.png)

#### 使い方
unity_chanという名前のPrefabがありますので、これをScene上において再生すれば、Animatorで設定されたデフォルトのアニメーションが再生されます。
PrefabのAnimationController.csを使ってアニメーションを切り替えていますので、適当にボタンでも追いて利用をしてみてください。
